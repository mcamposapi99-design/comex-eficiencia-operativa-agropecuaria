# 📊 Monitor de Eficiencia Agroexportadora - Argentina

## 🎯 Objetivo del Proyecto
Desarrollar una solución de **Business Intelligence** que articule la performance operativa del campo con el impacto económico real. El foco está en identificar la "Frontera de Eficiencia" por provincia, cruzando rendimientos físicos con liquidación de divisas (FOB).

## 🏢 Contexto de Negocio
Como analista con experiencia en operaciones (**Fintech/Mercado Libre/Pirelli**) y formación en **Economía**, busqué responder: 
*¿Qué provincias generan mayor densidad de valor (USD/Ha) y dónde existen brechas de ejecución operativa?*

## 🛠️ Stack Tecnológico
- **BigQuery:** Procesamiento de grandes volúmenes de datos.
- **SQL Avanzado:** Uso de CTEs y Window Functions para cálculo de participaciones.
- **Looker Studio:** Visualización interactiva y dashboards de performance.

## 📈 Insights Clave del Dashboard
- **Análisis de Ratio:** Implementación de métricas de USD por Hectárea.
- **KPI Operativo:** Ratio de ejecución (Superficie Cosechada vs. Sembrada).
- **Benchmarking:** Comparativa de rinde promedio vs. potencial máximo regional.
